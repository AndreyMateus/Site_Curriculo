# Site_Curriculo
Este código é um exemplo de um site pessoal, criado para um programador Jr .NET. Ele consiste em uma única página com várias seções, cada uma representando informações diferentes.

O site é totalmente responsivo, com um menu de navegação que pode ser acessado em dispositivos móveis, permitindo ao usuário navegar facilmente pelas diferentes seções do site. O menu contém links para seções como Início, Sobre, Currículo, Portfolio e Contato.

A primeira seção é a Hero Section, que contém informações sobre o criador do site, como nome, tecnologias que ele domina e sua posição como programador Jr .NET. Também há ícones de redes sociais, como GitHub, Instagram e LinkedIn.

A segunda seção é a Sobre, que contém uma breve descrição sobre o programador, suas habilidades e experiências.

A terceira seção é a Skills Section, que mostra as habilidades do programador em uma barra de progresso, como HTML, CSS e Bootstrap.

O site é totalmente construído com tecnologias da web, como HTML, CSS e JavaScript. Ele também usa algumas bibliotecas de terceiros, como Bootstrap, AOS e Typed.js.
